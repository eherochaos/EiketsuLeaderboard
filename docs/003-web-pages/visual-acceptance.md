# 视觉验收表

- 必须在 1440x900 检查首屏。
- 顶部导航必须可见。
- 主标题必须可见。
- 当前第一必须可见。
- Top 3 必须可见。
- 筛选入口必须至少部分可见。
- 核心数据禁止被挤到首屏外。
- 禁止出现大面积无意义空白。
- 主容器必须统一宽度。
- 所有 section 左边缘必须对齐。
- 同组卡片必须对齐。
- 表格列必须整齐。
- 指标列必须可横向比较。
- 间距必须来自 spacing scale。
- 相似模块间距必须一致。
- 颜色必须来自 CSS 变量。
- 禁止新增随意 hex 色值。
- CTA 颜色必须统一。
- 页面必须只有一个最强主标题。
- 数据数字必须比标签醒目。
- 重复组件样式必须一致。
- loading、empty、error 必须有明确状态。
- 图表必须能快速读出结论。
- 禁止图表只好看但难读。
- 移动端必须无横向滚动。
- 最终回复必须包含“视觉自检”结果。
