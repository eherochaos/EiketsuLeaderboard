页面设计不准许搞模糊空间，任何约束都必须固定

## 1. 首屏信息密度
在设计文档中必须明确以下内容
- 首屏必须出现什么
- 哪些信息不能被挤到首屏外
- Hero最大高度
- 连续空白最大高度
- 主 CTA 是否可见
- 核心数据是否可见

## 2. 页面网格 / 对齐
在设计文档中必须明确以下内容
- 主容器宽度
- 左右 padding
- section 对齐方式
- 两栏比例
- 列表 grid 结构
- 表格列宽
- 指标列固定宽度

## 3. 间距系统
- 只允许使用 4 / 8 / 12 / 16 / 24 / 32 / 48
- section 间距是多少
- 卡片 padding 是多少
- 标题到说明是多少
- 说明到内容是多少
- 列表行距是多少

## 4. 色彩系统 / 配色表
--color-bg          页面背景
--color-panel       卡片背景
--color-text        主文字
--color-muted       次级文字
--color-brown       顶部导航 / 品牌结构
--color-gold        强调 / 激活 / 分割
--color-red         主 CTA
--color-border      边框

## 5. 字体系统
- 主标题字号
- section 标题字号
- 正文字号
- 辅助文字字号
- 数据数字字号
- 字重
- 行高
- 字间距

## 6. 组件设计
- 至少要有这些组件规范：
Button
Card
Tag / Badge
Metric
Table
Tabs
Filter Bar
Deck Row
Chart Card
Modal / Drawer
Empty State
Loading State
Error State

- 每个组件都要规定：
尺寸
padding
圆角
颜色
边框
hover
active
disabled
loading
错误态
可不可以嵌套


## 7 数据展示规则
所有需要展示的数据必须展示规则

## 8. 图表规范
允许什么图表
禁止什么图表
颜色如何使用
图例放哪里
百分比怎么标
是否允许 3D / 饼图 / 装饰图

hover

展开 / 收起
排序中
筛选中
