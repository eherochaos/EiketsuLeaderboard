# 003 Web Pages

这是 Eiketsu Leaderboard 的网页界面说明，覆盖首页和 TierList 两个页面。

## 首页
- 顶部是品牌栏和 TierList 入口。
- 首屏展示环境数据概览、更新时间、主导势力、当前第一卡组和 Top3 卡组。
- 首屏下方有环境状态栏，展示正常、过热中、无数据、缺乏新构筑几种状态。
- 势力占比使用横向堆叠条展示环境构成。
- 代表卡组展示排名、主卡头像、势力、卡组名、核心构成和关键指标。
- 注目单卡展示卡图、使用率、胜率、综合 Rank 趋势和指标标签。
- 页面底部说明字段来源。

## TierList
- 顶部展示榜单名称、更新时间、样本数和当前第一。
- 筛选区提供势力、命名来源和排序项。
- 桌面端使用表格展示榜单。
- 手机端使用榜单行块展示榜单。
- 每条榜单展示 Rank、卡组名、分类名、胜率、使用率、倾奇点、样本数和详情入口。

## 视觉感受
- 整体是战术情报 Dashboard。
- 背景为米白纸感。
- 顶部导航为 `#24150e` 深棕。
- 重点操作使用 `#a83b34` 暗红按钮。
- 数据强调使用 `#b98524` 金色和官方势力色。
- 卡组和单卡图片用于读榜，不作为装饰主体。
